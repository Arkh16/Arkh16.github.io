What are registers?

Registers are data storage units which store 16 bits of data. Registers can be general-purpose or specific-purpose. General-purpose registers have no purpose and are used for simply whatever they are needed for. Specific-purpose registers have special uses and shouldn't be set to.

Flags

The FLG register contains bits that instruct the CPU on how to handle certain operations, and also store the state of previous instruction(s).
A list of flags (and their aliases) is below:

Bits are ordered from right to left.
Bit Alias Function
0 HALT Signals the processor to cease execution.
1 EQ Set if a comparator finds both operands to be equa.l
2 LT Set if a comparator finds first operand to be greater than the second.
3 GT Set if a comparator finds first operand to be lesser than the second.
4 QZ Set if a comparator finds first operand to be 0.
5 EDI If set, user-defined interrupts described in the IDT will be processed.
